# ჯაშუშის თამაში - Spy Game

მობილური თამაში აგებულია React Native და Expo-ს გამოყენებით.

## აღწერა

ჯაშუშის თამაში არის სოციალური დედუქციის თამაში, სადაც ერთი მოთამაშე არის ჯაშუში, ხოლო დანარჩენები იმყოფებიან კონკრეტულ ლოკაციაზე. ჯაშუშმა უნდა გამოიცნოს ლოკაცია, ხოლო დანარჩენებმა უნდა აღმოაჩინონ ჯაშუში.

## თამაშის მახასიათებლები

- 🎮 მინიმუმ 3 მოთამაშე
- 🕵️ ერთი შემთხვევითი ჯაშუში
- 📍 შემთხვევითი ლოკაცია
- ⏱️ ჩართული ტაიმერი დისკუსიისთვის
- 🌙 მუქი თემა
- 🎭 როლების გამოცხადების ანიმაცია

## როგორ ვითამაშოთ

1. აირჩიეთ მოთამაშეების რაოდენობა
2. დააჭირეთ "თამაშის დაწყებას"
3. თითოეული მოთამაშე ნახავს თავის როლს:
   - ჯაშუში: უნდა გამოიცნოს ლოკაცია
   - მოქალაქე: იცის ლოკაცია და უნდა იპოვოს ჯაშუში
4. დისკუსიის დროს გამოიყენეთ ტაიმერი
5. ბოლოს მოხდება კენჭისყრა ჯაშუშის გამოსავლენად

## ტექნიკური დეტალები

აგებულია შემდეგი ტექნოლოგიების გამოყენებით:

- ⚛️ React Native
- 📱 Expo
- 🎨 React Native Paper (UI კომპონენტები)
- 💫 React Native Reanimated (ანიმაციები)
- 🔒 TypeScript
- 🎯 საუკეთესო პრაქტიკები და მუდმივი ოპტიმიზაცია
